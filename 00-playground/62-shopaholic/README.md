# Shopaholic

A shopping list app where users can add/delete items and manage a "favs" list. Favorite items can be bulk imported into the main shopping list.

## Features

- setting up Copilot in VS Code.
- trying Vibe Coding with Bolt.
- using edit and agent modes.
- using Copilot context.
- building a Next.js site with Copilot.
- writing instruction files: [global](copilot-instructions.md) and [per-extension](instructions/tsx.instructions.md).
- relying on git for version control.
- accessing Supabase through an MCP server.
- maintaining [a feature list](_instructions.md).

Based on [Coding with AI (Copilot) Tutorial](https://www.youtube.com/playlist?list=PL4cUxeGkcC9joeiiVaLExvfSgmdtBbSPM) by Shaun Pelling - Net Ninja (2025).
